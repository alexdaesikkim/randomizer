API Address: http://rdm-alpha.herokuapp.com/api/alpha/
  • GET
    Grab Random Song
    Entry Point:
    GET /random/{game}/{version}/

    Query Parameters (via object):
    Name Type Default Value Description
    count Integer 1 Number of songs to return.
    min_difficulty Integer 0 Minimum difficulty. Based on numbers, not string.
    max_difficulty Integer 0 Max difficulty. Based on numbers, not string.
    min_level Integer 0 Max level cap on the song returned
    max_level Integer 0 Max level cap on the song returned
    build String "latest" Game build. Find accepted values via API call to game's information.
    style String "single" Play style. Find accepted values via API call to game's information.
    na_option Boolean "single" NA Option. Filters songs not available in North American version of the game (Limited to DDR Ace only).

    Due to few inconsistencies across games, here are few game-specific options. Please note these notes if you plan to use one of these games.

    crossbeats REV.
    The return parameter "version" is not available.
    GROOVE COASTER
    The return parameter "version" is not available.

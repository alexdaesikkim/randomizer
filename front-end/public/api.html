Random Song
  • GET
    /api/alpha/random/{game}/{version}/
    Grabs random songs for given game/version.

    Query Parameters (via object):
    Name Type Default Description
    count Integer 1 Number of songs to return.
    min_difficulty Integer 0 Minimum difficulty. Based on numbers, not string.
    max_difficulty Integer 0 Max difficulty. Based on numbers, not string.
    min_level Integer 0 Max level cap on the song returned
    max_level Integer 0 Max level cap on the song returned
    build String "latest" Game build. Find accepted values via API call to game's information.
    style String "single" Play style. Find accepted values via API call to game's information.
    na_option Boolean "single" NA Option. Filters songs not available in North American version of the game (Limited to DDR Ace only).

    Due to few inconsistencies across games, here are few game-specific options. Please note if you plan to use one of these games.

    • crossbeats REV. (all version)
      The return parameter "version" is blank. The sort folder used to sort the song is stored under 'genre'.
    • GROOVE COASTER (all version)
      The return parameter "version" is blank. The sort folder used to sort the song is stored under 'genre'.
    • Dance Evolution ARCADE
      Game only features one difficulty, as separate difficulties within the game does not yield different levels.
    • DanceDanceRevolution EXTREME
      'versions' represent the sorting folder exisitng in game; it may not necessarily reflect which version of DDR it came from.
